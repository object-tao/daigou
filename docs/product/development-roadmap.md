# DropPilot 開發路線

## Phase 0: 基礎設施

- Cloudflare Workers + Assets + D1 部署。
- GitHub Actions CI/CD。
- 文檔與運行代碼目錄隔離。
- 需求、MVP、架構決策、上線清單文檔。

## Phase 1: 人工運營閉環

- 會員註冊、登入、資料、標籤、五級會員。
- 代購購物車與人工報價。
- Yahoo 代拍提交、授權上限、人工出價記錄。
- 銀行轉帳充值、人工入帳、餘額扣款。
- 入庫包裹、無主公海、合箱、一件直發、出庫。
- 增值服務：全方面拍照、加強包裝、抽空、議價。
- 郵件通知與 WhatsApp 付費偏好。
- 售後申請、退款/取消人工審核。
- 後台角色權限、員工帳號、操作日誌。

## Phase 2: 增長與報表

- 團購活動、推薦佣金、開團者獎勵。
- 代購商品積分、物流費用積分、商品兌換。
- 優惠券規則與使用限制。
- CSV/Excel 導出、財務/倉庫/客服報表。
- 多語內容管理：繁體中文、英文、日文。
- SEO 頁面結構與內容索引。

## Phase 3: 自動化與外部對接

- 日本平台 API 或半自動抓取。
- 支付網關：AlipayHK、微信、信用卡等。
- 自建物流追蹤事件與客戶查詢。
- OCR 與以圖搜圖。
- App 形態評估：PWA、Flutter、React Native。
- 舊系統資料遷移工具。

## 仍需業務確認

- 五級會員的升級門檻和運費優惠。
- 佣金百分比、可用範圍和撤銷規則。
- 免倉期天數與每日收費。
- 紙箱費、增值服務費、議價服務費。
- 團購開團者免費/獎勵的準確規則。
- 優惠券發放與使用限制。
- 舊系統資料結構與遷移批次。
