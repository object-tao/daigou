# DropPilot 需求整理

此目錄只放產品、架構、運維文檔，不參與前端或 Worker 打包。

## 產品定位

- 面向香港、澳門、台灣、英國、加拿大、澳大利亞的香港人，前期以繁體中文和港幣結算為主。
- 不做中國大陸用戶。
- 前期以人工代購、人工代拍、人工審核和人工匯率為主，一年後再評估日本平台 API 對接。

## 首期範圍

- H5 自適應會員端。
- 營運後台。
- 日本代購購物車，跨平台合併提交。
- Yahoo 拍賣人工代拍。
- 船橋倉作為最終出發集運倉，其他門市/收貨點可擴展。
- 銀行轉帳充值，人工審核入帳。
- 人工匯率設置，港幣折算日元成本。
- 五級會員、會員標籤、人工認證。
- 郵件通知，WhatsApp 作為付費附加通知。
- 操作日誌、角色權限、員工帳號、CSV/Excel 導出。

## 延後或待定

- Mercari 等平台 API 對接。
- 支付寶、AlipayHK、微信、信用卡、PayPay 等支付網關。
- 圖片搜索、OCR、以圖搜圖。
- APP 形態，後續在 Flutter、React Native、PWA 間評估。
- 優惠券詳細規則。
- SLA、維護範圍和舊系統數據遷移明細。
